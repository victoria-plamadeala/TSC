InkTime - Smartwatch Project

Descriere Generala:
Acest proiect vizeaza designul si integrarea unui smartwatch bazat pe nRF52840, utilizand un display e-paper de 1.54 inch pentru eficienta energetica. Dispozitivul include circuite de management al alimentarii pentru o baterie Li-Po de 250mAh si un motor haptic pentru notificari.

Detalii de Fabricatie si Asamblare:
PCB-ul este optimizat pentru productie industriala, avand trasee de alimentare de 0.3 mm si trasee de date de 0.15 mm. Ansamblul mecanic a fost verificat prin modelare 3D, asigurand integrarea componentelor (baterie, display, placa) in carcasa dedicata InkTime. Toate fisierele necesare pentru productie, inclusiv Gerber, BOM si coordonatele CPL, sunt disponibile in folderul 
/Manufacturing.
